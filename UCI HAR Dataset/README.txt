Эксперименты проводились с группой из 30 добровольцев в возрасте от 19 до 48 лет. Каждый человек выполнял шесть действий (ХОДЬБА, ПОДЪЕМ ПО ЛЕСТНИЦЕ, СПУСК ПО ЛЕСТНИЦЕ, СИДЕНИЕ, СТОЯНИЕ, ЛЕЖАНИЕ) с смартфоном (Samsung Galaxy S II) на поясе. Используя встроенный акселерометр и гироскоп, мы фиксировали трехосевое линейное ускорение и трехосевую угловую скорость с постоянной частотой 50 Гц. Эксперименты были записаны на видео для ручной маркировки данных. Полученный набор данных был случайным образом разделен на два набора, где 70% добровольцев были выбраны для создания обучающих данных и 30% - тестовых данных.

Сигналы датчиков (акселерометра и гироскопа) были предварительно обработаны путем применения шумовых фильтров, а затем выборки в скользящих окнах фиксированной ширины 2,56 секунды с перекрытием 50% (128 измерений/окно). Сигнал ускорения датчика, который имеет компоненты гравитационного и телесного движения, был разделен с использованием фильтра нижних частот Баттерворта на ускорение тела и гравитацию. Предполагается, что гравитационная сила имеет только низкочастотные компоненты, поэтому использовался фильтр с частотой отсечки 0,3 Гц. Из каждого окна был получен вектор признаков путем вычисления переменных из временной и частотной областей. Более подробную информацию см. в файле 'features_info.txt'.

Для каждой записи предоставляется:

======================================

Трехосевое ускорение от акселерометра (общее ускорение) и расчетное ускорение тела.

Трехосевая угловая скорость от гироскопа.

561-мерный вектор признаков с переменными временной и частотной областей.

Метка активности.

Идентификатор субъекта, проводившего эксперимент.

Набор данных включает следующие файлы:

=========================================

'README.txt'

'features_info.txt': Содержит информацию о переменных, используемых в векторе признаков.

'features.txt': Список всех признаков.

'activity_labels.txt': Связывает метки классов с названиями действий.

'train/X_train.txt': Обучающий набор.

'train/y_train.txt': Метки обучения.

'test/X_test.txt': Тестовый набор.

'test/y_test.txt': Тестовые метки.

Следующие файлы доступны для обучающих и тестовых данных. Их описания эквивалентны.

'train/subject_train.txt': Каждая строка идентифицирует субъекта, который выполнял действие для каждого образца окна. Диапазон от 1 до 30.

'train/Inertial Signals/total_acc_x_train.txt': Сигнал ускорения от акселерометра смартфона по оси X в стандартных единицах силы тяжести 'g'. Каждая строка показывает вектор из 128 элементов. То же описание применимо к файлам 'total_acc_y_train.txt' и 'total_acc_z_train.txt' для осей Y и Z.

'train/Inertial Signals/body_acc_x_train.txt': Сигнал ускорения тела, полученный путем вычитания силы тяжести из общего ускорения.

'train/Inertial Signals/body_gyro_x_train.txt': Вектор угловой скорости, измеренный гироскопом для каждого образца окна. Единицы измерения - радианы/секунду.

Примечания:

======

Признаки нормализованы и ограничены в диапазоне [-1,1].

Каждый вектор признаков представляет собой строку в текстовом файле.

Единицы измерения, используемые для ускорений (общего и тела), - 'g' (гравитация земли -> 9,80665 м/сек?).

Единицы измерения гироскопа - рад/сек.

Видео эксперимента, включающее пример 6 записанных действий с одним из участников, можно увидеть по следующей ссылке: http://www.youtube.com/watch?v=XOEN9W05_4A
